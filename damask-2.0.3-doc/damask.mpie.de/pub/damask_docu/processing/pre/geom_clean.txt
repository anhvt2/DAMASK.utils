---++ Purpose
Smooth geometry by selecting most frequent microstructure index within given stencil at each location.

%UPTOC%
---++ Usage
<pre>
&gt; geom_clean options <a href=https://https://damask.mpie.de/Documentation/GeometryFormat>Geom file</a>(s) 
</pre>

%UPTOC%
---++ Options
   $=-s= / =--stencil= [ *3* ]: 
   size of smoothing stencil
