---++ Purpose
Unpack geometry files containing ranges "a to b" and/or "n of x" multiples (exclusively in one line).

%UPTOC%
---++ Usage
<pre>
&gt; geom_unpack options file[s] 
</pre>

%UPTOC%
---++ Options
   $=-1= / =--onedimensional=: 
   output geom file with one-dimensional data arrangement
